# todo

## Installation

* `git clone https://github.com/<this-repository>`
* `cd todo`
* `npm install`

## Running / Development

* `lux db:reset`
* `lux db:migrate`
* `lux db:seed`
* `lux serve`

## Testing

* `lux test`
